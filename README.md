## Getting Started

```shell
git clone https://github.com/Rintaro-Fukui/Calendar-Generator.git
cd ./Calendar-Generator/app
python3 -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```
